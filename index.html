Tampermonkey

Tampermonkey é uma extensão grátis para seu navegador e o gerenciador mais popular de scripts de usuário para navegadores baseados em Blink e WebKit, tais como Chrome，Microsoft Edge，Safari，Opera Next e Firefox.

Mesmo que alguns dos navegadores suportados tenham suporte nativo a scripts de usuário, o Tampermonkey lhe dá muito mais comodidade na gestão de seus scripts. Ele fornece recursos como a instalação fácil de scripts, verificação automática de atualizações, uma visão geral simples de que scripts estão rodando em um guia, um editor embutido e há uma boa chance de que os scripts incompatíveis rodem bem no Tampermonkey.

  • Todos os script apresentados nesse site pode ser executador no Tampermonkey
